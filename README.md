# coolwm
A very simple stacking window manager for X11, inspired by cwm and tinywm.  

##Current Features:

|Command|Keybinding|
|-------|----------|
|Launch xterm|Alt-Return|  
|Launch xclock|Alt-c|
|Move focused window by 20px|Alt-(h/j/k/l)|  
|Cycle through windows|Alt-Tab|
